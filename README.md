# Engineering Calculator v1.0

A **professional engineering calculator** with a modern user interface and a **Toggle Theme** feature.  
Built using **Python** and **CustomTkinter**, this app is ready to run on **Windows**.

---

## ⚙️ Features

- Modern GUI with **light and dark themes**  
- **Toggle Theme** button to switch between light and dark modes  
- Supports:
  - Basic arithmetic: addition, subtraction, multiplication, division  
  - Parentheses and operation precedence  
  - Decimal and integer numbers  
- Ready to be published as a **standalone Windows executable (.exe)**
